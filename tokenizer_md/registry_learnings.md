# Learnings Registry

Index of all library-specific learnings. When working with a component, find its library here and load the relevant file.

| Library | File | Used for |
|---|---|---|
| shadcn/ui | `learnings_shadcn.md` | UI components: button, card, tabs, separator, avatar, badge, etc. |
| react-resizable-panels | `learnings_resizable.md` | Resizable panel layouts |
| dnd-kit | `learnings_dndkit.md` | Drag and drop interactions |
| @xyflow/react | `learnings_xyflow.md` | Node-based diagrams, flow charts, canvas layouts |
| Tailwind CSS | `learnings_tailwind.md` | Styling, layout patterns, CSS gotchas |
| react-arborist | `learnings_arborist.md` | Tree list views with expand/collapse, drag-to-reorder |
